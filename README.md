# SDL-Pong
A simple pong game written in c++ using the SDL library
<br>
Controls:
<br>
  Left paddle up:    W
<br>
  Left paddle down:  D
<br>
  Right paddle up:   ↑
<br>
  Right paddle down: ↓
<br>
  Exit:              Esc
<br>
  On windows use build.bat to build
  On linux use build.sh to build
<br>
  Note: You must have the SDL library installed for both windows and linux
